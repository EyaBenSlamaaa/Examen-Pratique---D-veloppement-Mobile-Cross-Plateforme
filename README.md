# Atlas Géographique

Application Flutter affichant un atlas interactif des pays du monde.

## Fonctionnalités
- Page d'accueil
- Liste des pays
- Détails d’un pays (capitale, superficie, langue…)
- Menu Drawer
- Page à propos

## Architecture
- Flutter / Dart
- Assets : drapeaux et image d’accueil
